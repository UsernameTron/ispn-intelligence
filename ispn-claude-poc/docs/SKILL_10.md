# Financial Close Acceleration

## Overview

Month-end financial close automation system for multi-entity organizations (ISPN + iGLASS + DxTEL) reducing close cycle from 7 days to 4 days. Automates account reconciliations, multi-entity consolidation, variance analysis, and board reporting package generation with audit-ready documentation.

**Primary Use:** Month-end close process automation, financial consolidation, variance analysis, board reporting
**Key Value:** Reduces close from 7 days to 4 days, saving 36 hours/month while improving accuracy
**Annual Impact:** $227,000+ in CFO/Controller time savings and faster financial reporting

## When to Use This Skill

- Monthly close process (day 1-4 of following month)
- Multi-entity consolidation (ISPN + iGLASS + DxTEL)
- Variance analysis and management reporting
- Board financial package preparation
- Quarter-end and year-end close (extended process)
- Audit preparation and documentation

**Request:** "Execute month-end close for November 2024. Consolidate ISPN, iGLASS, and DxTEL financials, perform reconciliations, generate variance analysis, and create board-ready financial package."

## Core Capabilities

### 1. Automated Reconciliations

**What Claude Automates:**
- Bank reconciliations (3 entities x 4 accounts = 12 reconciliations)
- Intercompany eliminations (between ISPN, iGLASS, DxTEL)
- Revenue reconciliation to customer invoices
- AR aging analysis and reserve calculations
- AP accruals and cut-off testing
- Prepaid/deferred revenue roll-forwards
- Fixed asset depreciation schedules

**Reconciliation Workflow:**
1. Pull transaction data from accounting system export
2. Match transactions to supporting documentation
3. Identify exceptions and variances
4. Generate reconciliation workpapers
5. Flag items requiring CFO review
6. Document audit trail

### 2. Multi-Entity Consolidation

**Consolidation Process:**
- Entity-level P&L: ISPN, iGLASS, DxTEL (separate)
- Intercompany eliminations (revenue/expense between entities)
- Consolidated P&L with segment reporting
- Balance sheet consolidation with eliminations
- Cash flow consolidation
- Management reporting by business unit
- Board reporting consolidated view

**Accuracy Checks:**
- Elimination entries tie out (revenue/payable matching)
- Balance sheet balances
- Cash reconciliation across entities
- Revenue recognition compliance (ASC 606)

### 3. Variance Analysis

**What Claude Analyzes:**
- Actual vs. Budget: Revenue, EBITDA, expense categories
- Month-over-month trends: Identify anomalies
- Year-over-year comparisons: Growth trends
- Driver analysis: Volume, pricing, mix impact on revenue
- Expense analysis: Fixed vs. variable, cost per customer
- Segment performance: ISPN vs. iGLASS vs. DxTEL

**Management Commentary:**
- Favorable variances: What drove outperformance?
- Unfavorable variances: Root cause and mitigation
- Trend analysis: Patterns requiring attention
- Forecast impact: Adjust forecast based on actuals

### 4. Board Financial Package

**What Claude Generates:**
- Executive summary memo (1 page)
- Consolidated P&L with variance commentary (2 pages)
- Balance sheet highlights (1 page)
- Cash flow summary (1 page)
- Segment performance (ISPN/iGLASS/DxTEL) (2 pages)
- KPI dashboard (1 page)
- Forward outlook and forecast (1 page)

**Package Quality:**
- Board-ready formatting and presentation
- Audit-quality workpaper documentation
- Exception-based reporting (highlight variances >5%)
- Professional commentary suitable for PE investors
