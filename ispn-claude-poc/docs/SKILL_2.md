## Best Practices

1. **Sunday Evening Executive Prep**: Generate comprehensive executive intelligence brief every Sunday at 6:00 PM to prepare for Monday leadership meeting and week ahead. Reduces Sunday prep time from 4 hours to 15 minutes.

2. **Daily Morning Flash**: Run quick portfolio health check every Monday morning (5 minutes) to identify emerging issues early and optimize CEO calendar for high-priority decisions.

3. **Weekly Board Preparation**: Create draft board materials first Monday of each month, allowing time for CEO review, refinement, and stakeholder pre-briefing. Reduces board prep from 12 hours to 1 hour.

4. **Real-Time M&A Integration Tracking**: Monitor iGLASS and DxTEL integration status daily during active integration periods (first 90 days), then weekly for ongoing optimization. Enables proactive intervention before issues escalate.

5. **M&A Pipeline Management**: Evaluate acquisition targets within 48 hours of opportunity identification to maintain competitive advantage in deal flow. Rapid evaluation capability differentiates ISPN in acquisition negotiations.

6. **Exception-Based Leadership**: Configure alerts for initiatives, acquisitions, or metrics falling below performance thresholds requiring CEO attention. Focus CEO time on strategic decisions, not routine monitoring.

## Common Questions

**Q: How does Claude identify which items require CEO escalation vs. can be delegated?**
A: Claude applies CEO-level escalation criteria based on: (1) Strategic value at risk >$250K, (2) Board or investor stakeholder involvement, (3) Cross-acquisition or cross-functional issues requiring CEO authority, (4) M&A decisions (all require CEO review), (5) Crisis situations with potential reputation or financial impact >$100K.

**Q: Can Claude analyze acquisition targets outside the filesystem (e.g., confidential data rooms)?**
A: Yes, download target materials to designated Filesystem MCP folder (e.g., `/CEO/M&A-Pipeline/[Target-Name]/`) and Claude will access for evaluation. All processing occurs locally through MCP - no data transmitted externally.

**Q: How does Claude track multiple simultaneous acquisitions (iGLASS, DxTEL, future targets)?**
A: Claude accesses Asana integration projects for each acquisition, consolidates status from multiple sources, and generates unified dashboard showing all integrations side-by-side with portfolio-level insights and capacity assessment.

**Q: What if board materials require confidential information?**
A: All processing occurs locally through MCP connections. No data transmitted to external services. Board materials can be generated with different confidentiality levels (internal vs. board vs. investor versions) by specifying in the request.

**Q: How does Claude help with PE investor relationships (Align Capital Partners)?**
A: Claude generates monthly investor update materials, tracks portfolio company performance against PE value creation plans, prepares for investor calls with briefing materials, and creates board presentations that align with PE governance requirements.

**Q: Can Claude support crisis response situations?**
A: Yes, use "crisis mode" command for rapid stakeholder analysis, impact assessment, response options, and multi-stakeholder communication package creation. Generates executive brief, board notification, investor update, and team communication in 15-20 minutes.

## Implementation Notes

### First-Time Setup:
1. Configure Asana MCP access to CEO and strategic initiative workspaces
2. Establish Filesystem MCP directory structure for board reports, financial data, M&A pipeline
3. Enable Slack MCP for leadership communication channels
4. Activate `pptx`, `xlsx`, `docx` skills for document generation
5. Create initial executive brief with current data to establish baseline
6. Customize escalation criteria and reporting preferences for CEO persona
7. Test board package creation with upcoming board meeting

### Scaling Considerations:
- Designed for CEO of $50M-$100M multi-acquisition organization
- Supports 2-4 active acquisitions simultaneously
- Handles board governance for PE-backed companies (Align Capital Partners model)
- Scales to enterprise organizations with 10+ acquisitions as portfolio complexity increases
- Requires standardized M&A integration playbook for optimal accuracy
