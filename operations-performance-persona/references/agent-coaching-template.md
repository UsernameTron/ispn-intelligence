## Coaching Intervention Library

### For High AHT (Talk Time)

**Call Control Training:**
1. Active listening techniques
2. Paraphrasing to confirm understanding
3. Asking clarifying questions early
4. Polite interruption techniques
5. Transitional phrases to move call forward

**Troubleshooting Efficiency:**
1. Decision tree usage
2. Symptom-based diagnosis
3. Elimination approach
4. Documentation while talking

### For High AHT (Hold Time)

**Knowledge Navigation:**
1. KB search techniques
2. Bookmark key articles
3. Use call types to predict resources
4. Pre-staging common tools

**System Proficiency:**
1. Keyboard shortcuts
2. Multi-window setup
3. Browser optimization
4. Tool-specific training

### For Low FCR

**Empowerment Awareness:**
1. Review resolution authority levels
2. When to escalate vs. resolve
3. Partner-specific policies
4. Exception handling guidelines

**Troubleshooting Depth:**
1. Root cause vs. symptom
2. Verification steps
3. Future prevention guidance
4. Documentation for future reference

### For Low Quality Scores

**Process Adherence:**
1. Call flow review
2. Required disclosures
3. Documentation standards
4. Hold/transfer protocols

**Calibration:**
1. Listen to exemplar calls
2. Self-evaluation exercise
3. Peer comparison discussion
4. QA criteria review

### For Negative Sentiment

**Soft Skills:**
1. Empathy statements
2. Tone awareness
3. Acknowledging frustration
4. De-escalation techniques

**Opening/Closing:**
1. Warm greeting
2. Agenda setting
3. Expectation management
4. Positive close with next steps

## Escalation Path

### If Coaching Not Effective (2 weeks):

1. Extend coaching period with increased monitoring
2. Document specific non-improvement areas
3. Consider peer mentor assignment
4. Supervisor escalation to Operations Manager

### If Continued Non-Improvement (4 weeks):

1. Formal performance improvement plan (PIP)
2. HR involvement
3. Daily monitoring
4. Clear improvement timeline with consequences

### If Critical Performance Issue:

1. Immediate supervisor consultation
2. HR notification same day
3. Consider temporary reduced queue access
4. Formal documentation required
